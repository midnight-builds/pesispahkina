# Pesäpallon junioripesiksen ikäluokat

Taustatieto: viralliset junioripesiksen ikäluokat. Ikä lasketaan kalenterivuoden
lopun mukaan (pelaaja kuuluu luokkaan sen mukaan, minkä ikäinen on vuoden lopussa).

| Luokka | Ikä           | Huom.                          |
|--------|---------------|--------------------------------|
| G      | alle 8 v      | nuorimmat, "tenavat"           |
| F      | 9–10 v        |                                |
| E      | 11–12 v       |                                |
| D      | 13–14 v       |                                |
| C      | 15–16 v       |                                |
| B      | 17–19 v (tytöt), 17–18 v (pojat) | |
| A      | 18–21 v (pojat) |                              |

PesisPähkinän ensimmäinen versio kattaa ikäluokat **G, F, E ja D**.

Kentät ikäluokittain (taustatietoa, ei välttämättä pelissä tarvittavaa):
- Tenavakenttä: E-tytöt sekä F- ja G-juniorit
- Juniorikenttä: E-pojat sekä D-tytöt
- Naisten kenttä: naiset, B- ja C-tytöt sekä C- ja D-pojat
